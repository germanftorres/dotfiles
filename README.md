# dotfiles used by germanftorres

## About

This dotfiles contain my configuration for my Linux dev environment, which at the
current time is mostly Elixir development under Windows trhough WSL (Windows Subssytem
for Linux).

A lot of blog posts and articles have been used as guidelines:

See [this post](https://www.atlassian.com/git/tutorials/dotfiles) on 
how to setup a backup system for dotfiles through git.

Also, I've been following this [fantastic article](https://www.mitchellhanberg.com/post/2018/10/18/how-to-use-elixir-ls-with-vim/)
on how to setup a LSP language server for elixir in vim.
